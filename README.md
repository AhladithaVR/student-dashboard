# Student Dashboard

A Student Dashboard web application built using Spring Boot that helps manage students, tasks, attendance, and user authentication.

## Features

- User Registration & Login (JWT Authentication)
- Student Management (Add, Update, Delete, View)
- Task Management
- Attendance Management
- Dashboard with statistics
- Responsive UI

## Technologies Used

- Java
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- HTML
- CSS
- JavaScript

## Prerequisites

- Java JDK
- Maven
- MySQL

## How to Run

1. Clone the repository.
2. Configure MySQL database.
3. Update database credentials.
4. Run:

```bash
mvn spring-boot:run
```
